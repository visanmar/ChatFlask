# Basic Python Flask chat

## Requirements and initialization

### Python virtual environment
```cmd
pip install virtualenv
python -m virtualenv -p python3 env
.\env\Scripts\activate
```

### Dependencies
```cmd
pip install flask
pip install python-socketio
```

### Execute
```cmd
python ./src/main.py
```
